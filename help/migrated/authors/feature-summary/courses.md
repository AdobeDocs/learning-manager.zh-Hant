# 創建、修改和發佈課程

若要瞭解如何在學習管理器中創建課程、認證和學習計劃，請閱讀本文。

作者可以創建學習物件，例如課程、認證和學習計劃。 學習者可以使用這些學習物件，而管理員可以跟蹤學習者的進度。

## 學習管理器中的課程 {#coursesincaptivateprime}

Adobe Systems學習管理員使作者能夠使用與虛擬培訓、自定進度培訓、課堂培訓和活動相關的一個或多個模組創建課程。 管理員可以進一步使用這些課程來創建課程實例、註冊學習者、分配鎖屏提醒以及為這些課程啟用反饋。 他們還可以使用這些課程創建學習計劃、學習計劃和認證。

作者可以使用通過任何在線學習工具創建的數位學習內容。 其他支持的課程格式包括視頻檔、PDF、文檔、文檔、PPT 和 PPTX。

## 建立課程 - 基礎工作流程 {#createacoursebasicworkflow}

要創建課程，追隨以下步驟：

1. 以作者身份登錄Adobe Systems學習管理器，因為只有作者有權創建課程。 現在，在快速入門 頁面上，按兩下 **[!UICONTROL Create Courses]**。
1. **在課程概述**&#x200B;頁面中，輸入課程名稱。現在，輸入此課程的簡短描述，該描述顯示在課程卡片上。 此說明不得超過 140 個字元。 然後輸入課程的詳細概述，該概述顯示在課程詳細資訊頁面上。 說明不得超過 1500 個字元。

   作為作者，您可以在將模組添加到課程時查看模組的說明。

1. 要使您的課程以其他語言提供，請按下頁面左上角的添加新 語言。 選擇要提供課程的一個或多個語言。 按兩下 **[!UICONTROL Save]**。 有關詳細信息，請參閱 [為不同語言](/help/migrated/authors/feature-summary/content-library.md)添加內容。
1. **修改課程設置**-

   1. 在課程設定 頁面，選擇課程的技能。 從技能下拉清單，選擇所需的技能。 然後，從“級別”下拉列表中清單，選擇所需的級別。
   1. 選擇課程技能、級別並設置技能的學分。 如果需要，添加更多技能。
   1. **從註冊類型**&#x200B;下拉清單，選擇註冊類型。

   以下是註冊類型：

   * **經理提名：** 只有經理可以提名這些課程。 學習者無法註冊這些類型的課程。
   * **經理批准：** 經理批准這些課程。 學習者可以註冊這些課程，但未經經理批准，他們不會直接註冊這些類型的課程。 當學習者註冊這些類型的課程時，會將通知 請求發送給經理。 經經理批准，這些課程將列為學習者註冊。
   * **自行註冊：** 學習者可以直接自行註冊這些類型的課程。

1. 要儲存變更，請按下 **[!UICONTROL Save]**。 要發佈課程，請按兩下 **[!UICONTROL Publish]**。

## 建立課程 - 進階 工作流程 {#createacourseadvancedworkflow}

1. 以作者身份登錄Adobe Systems學習管理器，因為只有作者有權創建課程。 現在，在快速入門 頁面上，按兩下 **[!UICONTROL Create Courses]**。
1. **在課程概述**&#x200B;頁面中，輸入課程名稱。現在，輸入此課程的簡短描述，該描述顯示在課程卡片上。 此說明不得超過 140 個字元。 然後輸入課程的詳細概述，該概述顯示在課程詳細資訊頁面上。 說明不得超過 1500 個字元。

1. **修改課程設置**-

   1. 在課程設定 頁面，選擇課程的技能。 從技能下拉清單，選擇所需的技能。 然後，從“級別”下拉列表中清單，選擇所需的級別。
   1. 選擇課程技能、級別並設置技能的學分。 如果需要，添加更多技能。
   1. 如果需要，將自定義合規性標籤添加到課程。 請參閱將 [合規性標籤添加到課程/學習路徑/認證](/help/migrated/authors/feature-summary/courses.md#add-compliance-labels-to-courselearning-pathcertification)。
   1. **從註冊類型**&#x200B;下拉清單，選擇註冊類型。

   以下是註冊類型：

   * **經理提名：** 只有經理可以提名這些課程。 學習者無法註冊這些類型的課程。
   * **經理批准：** 經理批准這些課程。 學習者可以註冊這些課程，但未經經理批准，他們不會直接註冊這些類型的課程。 當學習者註冊這些類型的課程時，會將通知 請求發送給經理。 經經理批准，這些課程將列為學習者註冊。
   * **自行註冊：** 學習者可以直接自行註冊這些類型的課程。

1. 選擇是要為課程設置價格還是使其免費。 如果要支付課程費用，請選擇該選項 **[!UICONTROL Paid]**，然後指定價格。 然後，價格會顯示在學習者的課程卡片和課程概述頁面上。

   注意：僅當配置了商務連接器時Adobe Systems才會啟用此功能。

1. 如果要為學習者提供取消註冊課程的功能，請選中「學習者可以自行&#x200B;**取消註冊」複選框**。

1. **實例配置**

   如果啟用此選項，則處於“進行中”狀態的學習者可以造訪其他實例並在那裡註冊。 然後，學習者可以保留上一個執行個體的進度。

   發佈課程后，如果返回到設定 頁面，則該選項將不再可編輯。

   您可以為以下課程類型開啟這個選項：

   * 自定進度
   * 課堂
   * 活動
   * 混合

   注意：複製課程時，如果您在源課程中啟用了實例配置選項，則該選項在目標課程中仍處於禁用狀態。

   **不支援實例切換**：

   * 付費課程
   * 經理提名的註冊類型課程。

   如果通過目錄共享實例交換機配置，則不會傳播到對等帳戶，該選項在目標課程中保持禁用狀態。

1. **多重註冊**

   使用此功能，您可以在一個或多個不同時期為學習者註冊多個課程執行個體。

   啟用切換多重 **註冊** 以在學習者的各種課程註冊之間切換。 如果已啟用實例切換，則無法使用多重註冊。

1. 選擇在上課之前必須完成的先決條件課程。 按兩下課程欄位，然後從課程清單中進行選擇。
1. **如果您希望將必修課程設為必填，請啟用「啟用****必要條件」**&#x200B;複選框。
1. 添加關鍵字作為與您的課程相關的標籤。 這些標籤可幫助學習者在搜尋期間輕鬆找到您的課程。 所有這些標籤都是根據我們添加的模組自動添加的。 如果您有其他標籤要添加到本課程中，您可以繼續輸入。
1. 添加關鍵字作為與您的課程相關的標籤。 這些標籤可幫助學習者在搜尋期間輕鬆找到您的課程。 所有這些標籤都是根據我們添加的模組自動添加的。 如果您有其他標籤要添加到本課程中，您可以繼續輸入。
1. 在自動停用欄位中，選擇課程停用的日期。 管理員必須先啟用自動停用選項。

## 無頭 用戶 介面的 RTF 描述支持

### 為什麼需要 CSS？

RTF 由 HTML 標記組成。 按原樣呈現標記將導致瀏覽器應用預設樣式。 這通常不符合公司的風格指南。 需要 CSS 才能符合準則。

### 默認樣式

附加的 CSS 樣式表包含學習管理器應用的樣式。 考慮到大多數用例，樣式進行了調整。 下載附加的 CSS 文件，並根據您的約定和版本編號系統將其導入您的 Web 應用程式。 定義的 CSS 類在 ql-編輯者 類下命名空間，它們不會干擾現有樣式。

### 自定義樣式

默認樣式可能無法滿足每個人的需求。 自訂可以通過覆蓋提供的 CSS 來完成。 所有樣式都包裝在 ql-編輯者 下作為後代選擇器。 使用以下類：

* 縮進： **li.ql-indent-$number**. $number從 1 到 9 不等
* 尺寸： **QL 大小小**， **QL 大小大**， **QL 大小巨大**

* 對齊方式： **ql-對齊-center**， **ql-align-justify**， **ql-align-right**

* 顏色： **QL-顏色-$color**。 $color = 白色、紅色、橙色、黃色、綠色、藍色、紫色
* 背景： **QL-BG-$color**。 $color = 黑色、紅色、橙色、黃色、綠色、藍色、紫色
* HTML 標籤： P， OL， UL， PRE， BLOCKQUOTE， H1， H2， H3， H4， H5， H6

[用於自定義的 CSS 檔案。](assets/ql-headless.css)

### API 變更以啟用呈現 RTF 概述

當客户版本編號無頭界面時，他們需要在他們正在開發的自定義用戶界面中顯示學習物件。 為此，通常會使用公開的 [GET /learningObjects](https://learningmanagereu.adobe.com/docs/primeapi/v2/#!/learning_object/get_learningObjects) API。 現在，學習管理器支援為概述欄位捕獲「富文本」，API 回應中學習對象的數據模型也會公開相同的內容。 請參閱以下 API 回應中模型片段中名為「richTextOverview」的字段。 另請注意，為了向後相容，之前公開的字段（“概述”）保持不變。

```
{ 
 "data": [ 
 { 
 "id": "string", 
 "type": "string", 
 "attributes": { 
 … 
 "localizedMetadata": [ 
 { 
 "description": "string", 
 "locale": "string", 
 "name": "string", 
 "overview": "string", 
 "richTextOverview": "string" 
 } 
 ], 
 … 
 }, 
 "relationships": { 
 … 
 } 
 } 
 } 
 ] 
} 
```

已在使用概述欄位的客戶在其無外設介面中不受影響，將像以前一樣只看到純文本。 如果客戶想要利用富文本概述，他們必須在 作者 UI 中為其學習對象創建格式豐富的概述，之後學習管理器除了 API 回應模型中的純文字（如以前）外，還將開始返回富文本概述。

不過，若要在 UI 中呈現此 RTF 文本，客戶將需要包含 CSS。 以下各節將對此進行詳細說明。

## Publish課程 {#publishacourse}

以學習者身份預覽課程后，您可以發佈課程，以便學習者可以使用該課程。 請注意，課程仍處於草稿模式。

典型的課程生命週期如下所示：

* **草稿** - 作者完成課程創建並保存時。 在此狀態下，課程尚不適用於學習者。
* **已發佈** - 當作者完成發佈課程時。 在此狀態下，學習者可以註冊該課程。 您也可以在此狀態下編輯課程。
* **已停用** - 發佈課程后，如果作者不希望課程顯示在學習者的課程目錄中，則作者可以將其移動到已停用狀態。
* **已刪除** - 處於已刪除狀態的課程是指從Adobe Systems學習管理器應用程式中完全刪除的課程。 只有作者才能刪除處於「草稿」或「已停用」狀態的課程。

![](assets/typical-course-lifecycle.png)

*課程生命週期的工作流*

要發佈您創建的課程，請按下 **[!UICONTROL Publish]** 頁面的右上角。

![](assets/publish-a-course.png)

*Publish課程*

在出現的確認快顯視窗消息中，按兩下 **[!UICONTROL OK]**。

該課程現在在課程目錄中可用。

## 常見問題 {#frequentlyaskedquestions}

+++如何刪除課程的「尋求經理提名」？

執行以下步驟：

1. 以作者身份登錄學習管理員。
1. 打開課程。
1. 在左窗格中，按兩下> **[!UICONTROL Settings]** **[!UICONTROL Edit]**。
1. **在「註冊類型**」下拉清單上，將註冊類型從&#x200B;**「經理提名」**&#x200B;更改為&#x200B;**「經理已批准**」或&#x200B;**「自行註冊**」。

1. 更改註冊類型后，重新發佈課程。

+++

+++如何組合課程？

您可以通過學習計劃合併課程。

1. 以管理員身份登錄學習管理員。
1. 在左窗格中，按兩下 **[!UICONTROL Learning Programs]**。
1. 要新增學習計劃，請按下 **[!UICONTROL Add]**。
1. 輸入學習計畫的詳細資訊，要儲存學習計劃，請按下 **[!UICONTROL Save]**。
1. 建立學習計畫後，按兩下 **[!UICONTROL Catalog]**。
1. 在課程卡片上，按兩下 **[!UICONTROL Add]**，如下所示。 對要添加到學習計劃的任意數量的課程重複此過程。

![](assets/add-catalog.png)

新增學習計畫中所需的所有課程後，按下 **[!UICONTROL Publish]**。

在學習方案中，您只能添加自註冊課程，而不能添加經理提名或經理批准的課程。 這是學習管理員中的預設行為。

+++

+++如何確保所有學習者都看不到所有課程？

您可以通過目錄實現此目的。 默認目錄包含預設情況下添加到學習管理器的所有課程。

您必須停用預設目錄並建立自訂目錄。

1. 以管理員身份登錄學習管理員。
1. 在左窗格中，按兩下 **[!UICONTROL Catalogs]**。
1. 按兩下 **[!UICONTROL Create]**&#x200B;建立目錄。 輸入詳細資訊並按下 **[!UICONTROL Save]**。

1. 在新創建的“目錄”選項上，您可以選擇可以添加的不同類型的學習，例如，學習計劃、認證或課程。
1. 在「學習計劃」部分中，按兩下 **[!UICONTROL Add Content]**。
1. 在左窗格中，按兩下 **[!UICONTROL Share Internally]** 或 **[!UICONTROL Share Externally]** 根據要目標對象。

1. 要新增使用者群組，請按下 **[!UICONTROL Add User Groups]**。
1. 在“目錄”頁面上，禁用 **D[!UICONTROL efault Catalog]**，然後啟用已創建的目錄。

![](assets/enable-custom-catalog.png)

+++

+++如何重新註冊已完成的課程？

課程完成不能回復。 學習者 **無法重新註冊** 已完成的課程。

+++

+++學習者在完成課程后如何視圖平均？

學習者可以在完成後通過按兩下課程中的重新訪問按鈕來視圖課程。

執行以下步驟：

1. 以學習者身份登錄。
1. 打開您已完成的課程。
1. 按兩下 **[!UICONTROL Revisit]**。

+++

+++如何在課程中添加資源檔？

創建課程時，您可以向課程添加與課程相關的視頻、音訊、pdf 或文本檔，以便學習者可以訪問其他培訓資料。

![](assets/add-resources.png)

+++

+++如何在模組上設置多次嘗試？

**先決條件：**&#x200B;管理員必須在“管理”應用程式中啟用“常規中的&#x200B;**多次嘗試**”選項&#x200B;**設定 >**“。

作為作者，在課程概述頁面中，啟用 **允許多次嘗試**&#x200B;選項。

有關詳細資訊，請參閱 [有關多次嘗試](courses.md#Allowmultipleattempts)的部分。

+++

+++您能否下載已上傳到學習管理器Adobe Systems內容以修改內容？

否，上傳到學習管理器的內容是已發佈的 zip 檔，而不是源檔。 因此，平均如果下載了內容，則無法在創作工具中编辑內容。 您需要源檔才能编辑內容。
