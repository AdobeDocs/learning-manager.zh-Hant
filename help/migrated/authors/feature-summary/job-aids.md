# 工作輔助工具

**工作輔助** 工具是一個培訓內容的資料庫，學習者無需註冊或完成標準即可取得。 學習者可以參考這些工作輔助工具，協助執行組織中的任何活動或任務。

工作輔助工具可獨立使用，或與課程共享一併使用學習管理器。

作者可以為學習者製作工作輔助工具。 利用工作輔助工具為學習者提供參考資料，如提示、檢查清單、指南，讓他們能持續使用以完成任務。

## 建立就業援助 {#createjobaid}

1. 在作者登入時，在左側選區選擇 **[!UICONTROL Job Aids]** 。
1. 在出現的頁面右上角選擇 **[!UICONTROL Create]** 。
1. 輸入姓名、描述和標籤。 選擇技能和相關等級。 如果你不希望其他作者使用工作輔助資料並分配給各自課程，請選擇內容為私人。

   只有現有技能才能用於工作輔助工具。 技能不是必須的。

1. 請在內容區上傳工作說明內容。

   影片、PDF、pptx 和 docx 是支援上傳的檔案格式。 專案壓縮檔或任何互動內容不支援上傳。

1. 輸入工作輔助工具的持續時間（分鐘）。
1. 點擊 **[!UICONTROL Save]**。

   工作輔助工具已經出版。

## 新增不同語言的工作協助 {#addcontentfordifferentlanguages}

1. 若要新增不同語言的工作輔助工具，請選擇 **新增語言** 標籤，並選擇所需語言。 透過這種方式，你可以為內容加入多語言支援。

   ![](assets/add-new-languagetab.png)

   *新增內容語言*

1. 重新申請新語言的職缺說明上傳流程。
1. 如果你想移除某一種語言，請選擇分 **[!UICONTROL Add New Language]** 頁並清除你的選擇。

   完成更改後，選擇「儲存」。

## 支援工作輔助工具的類型 {#typesofsupportedjobaids}

以下是工作輔助工具所支援的檔案格式。

* PDF
* PPT
* PPTX
* XLS
* XLSX
* DOC
* DOCX
* 所有影片檔案格式

>[!NOTE]
>
>不支援壓縮檔和圖片檔。

## 多語言工作輔助工具

Adobe Learning Manager （ALM） 中的多語言工作輔助工具，讓作者與管理員能在同一工作輔助工具條目中，以多種語言提供支持文件、指南或資源。 不同地區的學習者都能以偏好語言取得相關教材，提升理解力、合規性及使用者體驗。

**使用案例**

* 全球勞動力賦能：向多元勞動力提供安全手冊、流程指南或參考文件，提供多語言版本。
* 法規遵循：確保所有員工都收到相同的母語合規文件。
* 持續的入職流程：為全球新進員工提供當地語言的入職清單或常見問題。
* 減少重複：將所有語言版本的工作說明整合於單一條目中，簡化更新與報告。

### 主要特色

* 多語言支援：在單一工作說明中，為每種支援語言附上獨特的檔案或網址。
* 本地化名稱與說明：請以每種語言輸入工作輔助的名稱與說明。
* 統一管理：從一處編輯、更新並回報所有語言版本。
* 向下相容性：現有的單語言工作輔助工具會自動在所有新增語言間複製，直到新檔案上傳。

### 建立多語言工作輔助工具

1. 進入作者角色，選擇工作輔助工具。
2. 選擇建立工作協助。
3. 請以預設語言輸入工作說明的名稱與描述。
4. 新增預設語言的主要內容檔案或網址。
5. 省省工作援助。

### 新增語言

1. 在工作輔助編輯器中，選擇新增語言。
2. 從列表中選擇所需的語言。
3. 對於每新增的語言：
   * 輸入當地化的名稱和描述。
   * 上傳對應的內容檔案或提供特定語言的網址。
4. 所有必修語言都重複這個步驟。

### 編輯與管理語言

1. 若要更新特定語言的檔案或描述，請選擇語言標籤並視需要進行修改。
2. 若在工作輔助發布後新增語言，原始檔案會自動分配到新語言，直到上傳唯一檔案為止。
3. 可依需求移除或替換任何語言的檔案。

### 發表與學習體驗

1. 在所有語言和檔案加入後，發布工作說明。
2. 學習者會以他們選擇的內容語言看到工作說明，並附上相應的檔案或網址。
3. 若無法提供學習者的語言，則會顯示預設語言檔案。

### 報導

* 下載工作協助報告，查看所有與每個工作協助工具相關的檔案及語言細節。
* 報告內容包括語言、檔名及用於追蹤的使用資料。

### 最佳實務

* 提供姓名、描述及內容檔案的準確翻譯。
* 定期檢視並更新檔案，以確保跨語言的一致性。
* 使用明確的命名規則來區分不同語言的檔案。
* 透過切換內容語言來測試學習者體驗，以驗證檔案的正確傳遞。

多語言就業輔助工具讓您能在一次條目中向全球受眾提供支持資源，減少重複，並確保每位學習者都能以偏好語言獲得正確資訊。 此功能提升了 Adobe Learning Manager 的無障礙性、合規性及行政效率。

## 撤回/重新發布工作輔助工具 {#withdrawrepublishjobaids}

你可以點擊工作說明書旁邊的設定圖示，並選擇撤回，來撤回已發表的工作協助。

![](assets/job-aid-withdraw.png)

*編輯、撤回或預覽已發表的求職助學金*

點擊撤回的職缺分頁查看已撤回的就業輔助工具。 你可以點擊設定圖示並選擇發佈，重新發布已撤回的職缺。

## 工作輔助工具中對 HTML 套件的支援

工作輔助工具現在支援標準 HTML 套件，作為一種新內容類型。 透過這項功能，學習者可以在工作援助播放器中開啟檢視並下載 HTML 套件。

在建立工作輔助工具時，作者可以上傳標準 HTML 套件及其他支援的檔案格式。

![](assets/html-job-aid.png)

*HTML 套件的支援*

HTML 套件必須包含以下內容：

* 一份Index.html檔案。
* Index.html檔案必須位於壓縮檔的根目錄中。

指定要上傳的內容為壓縮檔，裡面包含Index.html檔案。

所有內容、資源與資產必須在 HTML 套件中被引用，並可透過 Index.html 存取。

## 常見問題 {#frequentlyaskedquestions}

建立工作輔助工具後，你可以在建立課程時將工作輔助資料加入課程。
