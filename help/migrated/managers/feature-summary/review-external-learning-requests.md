# 以經理的身份檢視外部學習請求

當你團隊中的學習者在 Adobe Learning Manager 提交外部學習請求時，你會收到平台內通知。 您可以查看提交細節，批准或拒絕申請，並為學習者添加留言。

## 經理審查工作流程的運作方式

當學習者提交外部學習請求時，會發生以下情況：

1. 你會收到 **應用程式內通知** ，提示你檢視提交內容。 提交資料會出現在您的 **主管儀表板的外部學習** 標籤中。
2. 你打開一個提交，檢查所有欄位和任何上傳的文件作為證明，然後選擇 **批准** 或 **拒絕**。
3. 你可以新增 **一個複習留言** ，讓學習者在收到通知時看到。
4. 學習者會 **收到平台內通知** ，告知您的決定。

若您批准提交，外部學習活動將被加入 **管理員學習者成績單** ，並顯示在學習者的成績單記錄中。

## 審查並批准或拒絕投稿

1. 以管理員身份登入 Adobe Learning Manager。

2. 在左側導覽窗格選擇 **「外部學習** 」。

3. 在提交清單中，選擇你想審查的請求。 投稿依提交日期排序，最近的在最上方。

4. 請審閱完整投稿：

   - 標題、描述、日期、持續時間與評分

   - 管理員設定的任何自訂欄位

   - 如有提供，附上證明文件。 選擇附件以查看或下載

5. 選擇 **批准** 或 **拒絕**。

6. 在 **「評論評論** 」欄位，輸入學習者所需的任何筆記。 這是可選的，但建議在拒絕請求時這麼做，讓學習者了解該修正的地方。

7. 選擇 **提交**。

學習者會收到應用程式內的決定通知。 如果您已批准提交，該報告現在會出現在學習者成績單中。

## 管理您的投稿排隊

你的外部學習佇列會顯示所有來自你直屬下屬的待處理及過去提交。

**依狀態篩選**

使用 **狀態** 篩選器縮小範圍：

- **All**- 顯示所有投稿，不論狀態

- **等待審核 -** 只顯示提交等待審核的狀態

- **已批准——** 顯示你已經批准的投稿

- **拒絕——** 顯示你已拒絕的投稿

**搜尋與排序**

- 請使用 **搜尋** 欄位依學習者姓名查找提交資料。

- 投稿預設依提交日期排序，最新者排在最上方。

### 核准路由規則

預設情況下，外部學習提交會轉送給學習者的直接主管。 當學習者沒有直接指派經理時，適用以下規則：

| **學習者有經理** | **學習者本身就是管理者** | **服從路由至** |
|---------------------------|-------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| 是的 | 不 | 直接管理（預設情況） |
| 是的 | 是的 | 直接管理（預設情況） |
| 不 | 不 | 若根帳戶使用者有管理員角色，則為該用戶;否則，提交將自動審核。 |
| 不 | 是的 | 若根帳戶使用者有管理員角色，則為該使用者;否則，提交會被導向學習者。 |

如果你對特定學員的經理分配有疑問，請聯絡你的帳戶管理員。

## 外部學習報告與成績單變更

當學習者的外部學習提交在 Adobe Learning Manager 中獲准時，該活動會被加入報告系統，並同時出現在管理員學習者成績單及學習者成績單中。

### 外部學習如何呈現在學習者成績單中

**注意：** 啟用外部學習會為管理員學習者成績單新增以下欄位： **外部學習名稱**、 **完成評論**，以及每個自訂欄位的動態欄位。 自訂欄位欄位總是會出現在匯出的最後。 若學習者成績單資料會輸入自動報告或商業智慧工具，請確保這些流程已更新以處理額外的欄位。

只有 **核准** 的外部學習申請才會出現在成績單中。 處於 **等待核准** 或 **拒絕** 狀態的提交資料不包含在成績單匯出中。

行政學習者成績單與學習者成績單對外部學習職稱的處理方式不同：

- 在 **行政學習者成績單**&#x200B;中，外部學習標題會放在現有 **的 LP/Certification/Course** 欄位中，並保持欄位結構與其他學習活動類型一致。

- 在&#x200B;**學習者成績單**（學習者生成）中，模組欄位後&#x200B;**&#x200B;**&#x200B;立即新增一個稱為&#x200B;**外部學習名稱**&#x200B;的欄位。

管理員設定的自訂欄位會在提交通過後，以動態欄位形式出現在兩次逐字稿匯出的末尾。

管理員學習者成績單中外部學習列的日期篩選是基於 **完成日期，該日期**&#x200B;對應於核准日期。
