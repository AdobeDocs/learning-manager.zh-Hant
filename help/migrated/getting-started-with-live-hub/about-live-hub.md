# 關於 Live Hub

Live Hub 將虛擬教室原生帶入 Adobe Learning Manager，因此不必在學習管理系統（LMS）與會議工具間切換。 每堂課都會自動追蹤，豐富的課程儀表板會讓講師看到學習者的參與度，遠超過出席。

Live Hub 專為即時訓練打造，AI 代理會與講師一同工作。 他們會建議投票、偵測參與者問題並提出答案、直播分組討論室，並產生課程摘要，讓講師能專注於培訓。 對學習者而言，錄影會自動分割成可瀏覽的主題，使重播如同現場出席一樣結構化且實用。

Live Hub 協助講師提供引人入勝的虛擬訓練，並協助組織大規模執行互動式即時訓練。 Live Hub（測試版）將於即將於八月的 Adobe Learning Manager 版本中推出。
