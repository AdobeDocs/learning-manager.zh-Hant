# 帳戶設定

1. 按一下您的像片/帳戶旁右上角的下拉箭頭，然後選取&#x200B;**[!UICONTROL Profile Settings]**。
1. 在快顯對話方塊中，您可以在設定檔像片區域中將滑鼠游標停留並按一下「**[!UICONTROL Edit]**」來新增/變更像片。
1. 按一下下方的文字區域，新增/修改&#x200B;**關於我**&#x200B;內容。
1. 按一下&#x200B;**[!UICONTROL Save]**。
