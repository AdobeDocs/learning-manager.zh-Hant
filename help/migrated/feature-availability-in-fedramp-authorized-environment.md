# FedRAMP 授權環境中的功能可用性

## 簡介

作為 FedRAMP 安全與合規要求的一部分，Adobe Learning Manager 的以下功能目前尚未在 FedRAMP 授權的環境中得到支援。 這些限制旨在保護受管制資料、維持邊界控制，並維護強制性的稽核標準。 我們正積極追蹤這些限制，並會隨著FedRAMP態勢演變，通報未來可用性的任何變動。
