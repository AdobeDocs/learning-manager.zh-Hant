# Adobe Learning Manager 一鍵註冊

透過電子郵件、聊天或其他通訊平台，從管理員那裡接收深度連結，以報名培訓模組或課程。 點擊連結即可立即報名並啟動模組，無需額外步驟。

## 註冊學習者並引導他進入課程

請注意以下情境及其相應的動作。 作為學習者，如果你點擊傳送給你的深層連結，根據你所處的情境，可能會發生以下其中一種情況：

| 不號 | 劇本 | 動作 |
| --- | --- | --- |
| 1 | 單模組課程與無序課程 | 你會自動註冊，玩家會自動開啟。 你必須按播放才能播放影片。 |
| 2 | 多模組課程與無序課程 | 你會自動註冊，並進入連結到該深層連結的特定模組。 你必須按播放才能播放影片。 |
| 3 | 多模組課程與有序課程 | 你會在概覽頁面跳出訊息，提示你必須先完成前一門課程，才能開始目前的課程，否則如果還沒完成前一門課。 在這種情況下，你會自動註冊，並進入課程的第一個模組。 |
| 4 | 課程有先修條件 | 如果前置條件都完成，你就可以在 Player 裡啟動模組。 按下播放鍵播放影片。 若未完成，會出現以下訊息：「無法跳過先修課程。 請在開始本課程前完成所有先修課程。 |
| 5 | 有候補名單的課程 | 你會被註冊。 你會進入課程總覽頁面，並被加入候補名單。 你也會看到課程被列入候補名單的訊息。 |
| 6 | 如果你已經選修了產生深度連結的課程。 | 你會落在模組上，玩家就會起飛。 按下播放鍵播放影片。 |

## 認證狀態與深度連結行為

除了上述情境外，還有兩種與驗證（登入與登出）狀態相關的使用者行為。

* 如果你點擊深層連結，且未登入 Adobe Learning Manager，將會被導向登入頁面。 登入後，深層連結會導向該模組。

* 如果你點擊深層連結，且登入 Adobe Learning Manager，系統會直接導向該模組。

## 行動與網頁的沉浸式支援

以下情境適用於 Adobe Learning Manager 的行動版與網頁版。

* 如果您沒有 Adobe Learning Manager 的桌面版，且收到深層連結，點擊該連結將導向瀏覽器中的 Adobe Learning Manager 登入頁面（若你未登入網頁版或行動版）。
* 如果您沒有 Adobe Learning Manager 桌面版且收到深層連結，選擇該連結將直接導向瀏覽器中的 Adobe Learning Manager 模組（如果您已登入網頁版或行動版）。

## 瀏覽器限制自動播放

當你點擊深層連結時，如果你沒有登入，它會將你導向到模組或登入頁面。 如果模組是影片，播放器不會自動播放影片。 在所有情況下，瀏覽器都會阻止播放器自動播放影片。 你必須手動按下播放鍵才能播放影片。
