# 學習者適應性課程

## 簡介

Adobe Learning Manager 中的自適應課程是由作者根據你的學習需求，根據你的學習需求強制要求，並在管理員啟用此功能後，根據你的學習需求自訂課程。 欲了解更多資訊，請參閱 [適應性課程](/help/migrated/administrators/feature-summary/adaptive-course-admin.md)。

## 以學習者的身份參加適應性課程

當你在 Adobe Learning Manager 上自適應課程時，你會看到一組根據你所屬使用者群組的個人化模組。

### 當你開啟適應性課程時，你會看到什麼

當你開啟一個適應性課程時，你只會看到與你相關的模組。 分配給其他使用者群組的模組對你來說完全看不到。

你的完成進度條和「X of Y 模組已完成」的計數只反映你所需的模組。 你修同一門課的同事可能會看到不同的模組數量、必修與選修的組合不同，或是完全不同的內容組合，這些都取決於他們的使用者群組。

### 報名適應性課程

報名適應性課程的運作方式與報名任何課程相同。 你可以從目錄自行登記，或由管理員幫你登記。

![](assets/image_0010.png)

如果你在課程中沒有根據目前使用者群組顯示的模組，就無法註冊。 如果發生這種情況，請聯絡你的管理員。 這很可能代表你的使用者群組分配需要更新。

### 在播放器中導航模組

當你打開球場播放器時，左側的目錄只顯示你能看到的模組。 你可以使用導航按鈕依序在模組中移動（針對已排序的課程），或以任意順序進入任何模組（對於未排序的課程）。

屬於其他學習者使用者群組的模組不會顯示在您的目錄中。

### 了解你的完成要求

當你完成所有標示 **為必修**&#x200B;的模組時，課程即告完成。 選修模組不會阻礙你完成學業。 你可以完成、重溫，或完全跳過，都不會影響完成進度。

### 如果教室或虛擬教室課程已滿，會發生什麼事

若你看到教室或虛擬教室模組，但已無剩餘名額，你將被選為該課程並列入該課程的候補名單。 你仍然可以立即存取並完成所有其他模組。 只有完整場次暫停。

當你在等候名單上時：

* 該遊戲會顯示在你的課程播放器中，但你無法啟動或存取它。
* 你的整體課程進度會正常延續到其他模組。
* 當有空位並被移出候補名單時，你會收到通知。

一旦分配座位，該場次即可進入，你可以加入或完成。

### 當你的檔案在課程中途改變時會發生什麼事

你的適應性課程體驗與你的使用者群緊密相連。 例如，如果你在註冊期間使用者群組因角色或地點改變而改變，模組會自動更新。

* 如果你更新的使用者群組讓先前隱藏的模組對你可見，這些模組就會出現在你的課程中。 如果新檔案需要這些，則會加入你的完成要求。
* 如果你更新的使用者群組不再包含某個模組，該模組將從你的視圖中移除。 如果該模組原本是必修但不再需要，則會從你的完成數中移除。
* 你已經完成的任何模組都會被標記為已完成，即使你的使用者群組改變了。 除非有新可見的模組之前沒做過，否則你不需要重做已完成的工作。
* 你的開始日期、已完成模組的日期，以及整體進度紀錄，在個人檔案變更後都會保持完整。

### 完成課程後會發生什麼事

當你完成所有必修模組後，課程即被視為完成。 屆時你會獲得任何相關的徽章或遊戲化點數。

完成後，若使用者群組變動且新增必需模組可用，完成可能會被回滾，且可能需要完成這些額外模組。

當這種情況發生時：

* 你的原始完成紀錄會留在成績單中作為歷史紀錄。
* 課程重新開放時仍處於進行中狀態，僅顯示新的必修作業。
* 你只需要完成新要求的模組。
* 當你的完成度刷新被觸發時，你會收到通知。
