# 簡介

虛擬教練是 Adobe Learning Manager 中的一項變革性功能，旨在彌合理論知識與實際應用之間的關鍵落差。 與其僅依賴被動學習方法，不如透過影片、測驗和閱讀材料。 Virtual Coach 為學習者提供互動式、沉浸式的環境，讓他們練習高風險對話並學習軟實力。

把虛擬教練想像成「24/7 練習夥伴」和「溝通飛行模擬器」。 正如飛行員在操作真實飛機前會累積數千小時的飛行模擬器，員工也能在遇到真實客戶、客戶或團隊成員類似情況前，先與 AI 驅動的虛擬角色進行複雜對話。 虛擬教練會即時根據學習者的輸入調整，提供即時且數據驅動的回饋，涵蓋他們所說的內容（知識）以及表達方式（表達方式與風格）。
