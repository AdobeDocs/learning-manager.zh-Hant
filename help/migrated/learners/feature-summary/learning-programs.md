# 學習計畫

學習如何使用 Learning Manager 中的 Learner 應用程式來瀏覽和使用學習程式。

學習計畫是一組獨特設計的課程，滿足特定學習者的目標。 管理者為學習者設計這些學習計畫。 你可以以學習者的身份來看待這些學習計畫。

## 概觀 {#overview}

Adobe Learning Manager 讓學習者能夠存取課程和學習計畫。 學習者可以透過目錄瀏覽所有可用的學習課程，或是自行註冊自己選擇的學習課程。 學習者可在課程中查看所有已註冊課程及分配的課程。

>[!NOTE]
>
>學習者也可從蘋果商店安裝 Learning Manager iPad 應用程式，並從 Google Play 安裝 Learning Manager Android 應用程式，並在行動裝置上存取課程。 所有學習者角色的功能也都在應用程式中提供。 學習者也可離線上課，線上後即可無縫存取。

## 學習計畫 {#learningprogram}

學習計畫是一組獨特設計的課程，滿足特定學習者的目標。 您可以在「我的學習」頁面查看分配給你的學習計畫。

作者與管理員可以創建學習計畫。

## 觀看學習程式 {#viewinglearningprograms}

你可以以學習者身份查看所有可用的學習計畫清單。 學習物件小工具會顯示所有已註冊及分配給學習者的學習程式。 從左側窗格點選目錄，篩選學習計畫以查看所有可用的學習程式。

在&#x200B;**「我的學習**」頁面，你可以根據訓練時長&#x200B;**和**&#x200B;訓練形式&#x200B;**篩選你所**&#x200B;參加的訓練。

你可以從課程目錄頁面查看每門課程的成效百分比。 一個課程或學習計畫可以有多個實例或課程。 點擊課程或學習計畫以查看詳細資訊。 你也可以看到每個課程/學習計畫的截止日期。

## 搜尋課程與學習計畫 {#searchingcoursesandlearningprograms}

Adobe Learning Manager 讓您能更快找到心儀的課程。 你可以用以下方式搜尋你的學習計畫：

1. 使用搜尋欄位。 點擊頁面頂端顯示的搜尋欄。 輸入課程名稱或與課程相關的關鍵字，即可找到你的學習計畫。 你也可以用預設標籤搜尋，比如 Captivate、C、Java、HTML 等等。 標籤可以在搜尋欄位內搜尋，也就是說，標籤會在你輸入時顯示在搜尋欄位。
1. 學習者可根據課程完成狀態搜尋學習對象：全部、尚未開始、已完成、進行中。
1. 學習者可從目錄頁面使用篩選、排序、能力及類型進行搜尋。

你可以點擊「能力」並選擇，依能力搜尋。 你可以點擊「排序」並選擇可用選項來排序課程。

## 報名學習課程 {#enrollingforlearningprograms}

學習者透過以下可能方式註冊學習計畫：

1. 管理員/經理會根據組織需求，為部分學習者報名強制性學習計畫。
1. 學習者可以直接報名課程或學習計畫。 如果課程/學習計畫是自我報名型，學習者會立即註冊。

學習者可以根據自己的選擇，報名多種學習計畫。 推薦學習小工具顯示你所有已註冊/分配的課程、學習計畫、工作輔助工具或證照。 你可以點擊探索並進入學習物件頁面報名。

你也可以選擇點擊「探索」，瀏覽目錄頁面中列出的任何學習課程。 報名頁面會出現。 點擊頁面右上角的「註冊」，以包含課程到課程清單。

一個課程或學習計畫可以有多個實例或課程。 在目錄中，點擊課程/學習計畫的磁貼名稱以查看詳細資訊。 你可以根據每個課程/學習計畫的截止日期，看到註冊到課程/學習計畫實例。

## 完成學習課程 {#completinglearningprograms}

作為學習者，你可以根據特定學習計畫的設定完成學習計畫。 你必須完成所有課程才能完成某些學習計畫，而完成標準可能是完成 X 門課（Y）中的 X 門課才能完成學習計畫。

如果管理員設定的完成標準是 2 分（滿分 4 分），那麼當你完成兩門課中的一門時，進度條會顯示 50% 的完成度。 完成兩門課程後，會顯示學習計畫的成功完成。

你也可以再回頭參加學習計畫，選修選修課程。
