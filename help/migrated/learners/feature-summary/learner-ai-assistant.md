# 學習者適用的AI助理

適用於學習者的AI助理(Beta)可協助他們從指派的學習內容中快速找到答案，而不需瀏覽整個課程。 您可以用簡單的語言提出問題，並獲得準確、重點突出的回應，以及相關課程內容的來源連結。

>[!IMPORTANT]
>
>適用於學習者的AI助理目前以Beta版功能提供。 功能、支援的情境和限制可能會隨著功能的演變而改變。


## 什麼是學習者適用的AI助理

AI Assistant是Adobe Learning Manager中創造性的AI支援聊天夥伴，可運用您信任的學習內容提供快速、準確的答案。 其中包含引文，讓您一律知道資訊的來源。

### 功能

- **智慧型問題回答**
   - 單圈與多圈交談
   - 英文的自然語言理解
   - 衍生自課程、認證、學習路徑和工作輔助的答案
   - 在查詢模稜兩可時智慧型澄清問題

- **內容來源和引文**
   - 從支援的目錄中的可用資源擷取答案
   - 提供直接連結至原始資料的引文
   - 支援所有Learning Manager內容格式（靜態和互動式）：PDF、DOCX、PPTX、XLSX、音訊(MP3、WAV、M4A)、視訊(MP4、MOV、WMV)、HTML、SCORM 2004和SCORM 1.2

- **使用者體驗**
   - 可從所有學習者頁面存取側面板介面
   - 因應內容區域的回應式設計
   - 在瀏覽器工作階段中維護的聊天記錄
   - 新登入或重新整理頁面時清理狀態
   - 友善、清晰、有教育意義的音調

- **系統管理員控制項**
   - 在帳戶層級啟用或停用此功能
   - 選取要納入AI回應的目錄
   - 依照Adobe AI准則，接受使用條款的要求

## 支援的內容型別

AI助理從指派給您的學習內容中擷取資訊，包括：

- **檔案：** PDF、Word、PowerPoint、Excel、HTML
- **媒體：**&#x200B;音訊(MP3、WAV、M4A)、視訊(MP4、MOV、WMV)
- **互動式內容：** SCORM 1.2、SCORM 2004
- **學習物件型別：**&#x200B;課程、學習路徑、認證、工作輔助

Adobe使用受信任的服務安全地處理您的學習內容。

### 目錄和內容來源限制

AI小幫手僅使用管理員明確設定的&#x200B;**內部**&#x200B;目錄中的內容。

目前版本不支援下列內容來源：

- **已共用**&#x200B;目錄
- **已取得**&#x200B;個目錄
- **外部**&#x200B;目錄
- **預設**&#x200B;目錄
- 協力廠商內容程式庫（例如LinkedIn Learning或Go1）

如果您無法存取課程或工作輔助，AI Assistant將不會從該內容顯示資訊，也無法存取引用連結。

## 使用案例

### 技術學習者

Sarah是銷售工程師，學習顯示卡的相關資訊。 她需要快速瞭解技術規格和優點，以自信地回答客戶的問題。

AI小幫手可協助Sarah：

- 對複雜GPU架構的清晰技術說明
- 深入瞭解各種繪圖卡及其差異
- 範例說明，以便Sarah能將功能與真實世界的使用案例聯絡起來

### 客戶支援

Marcus是合作夥伴公司的支援專家。 他需要快速瞭解產品功能，以協助客戶而不需要升級至工程團隊。

AI助理可協助Marcus執行下列工作：

- 尋找相關支援內容以供經常詢問的客戶查詢
- 在初始答案不夠具體時詢問澄清問題
- 尋找相關疑難排解課程的建議以改善其技能

### 新員工入門

Jennifer剛加入公司，被大量的訓練材料所淹沒。 她需要一種不需要複習整個課程就能找到特定資訊的方法。

AI助理可協助Jennifer完成以下工作：

- 取得有關提交費用報表的逐步指南
- 探索有關公司原則的課程，而不瀏覽整個目錄
- 指導她完成課程的適當部分，而不用做她的影片觀賞時間

## AI助理如何使用內容

AI助理可以從您的學習內容中找到精確答案。 以下是運作方式。

### AI助理使用的內容

AI助理只會使用帳戶管理員啟用的學習內容來回答問題。 所選目錄中的內容已編制索引。

AI Assistant會分析您指派的學習內容，以產生重點突出的情境式回應：

- 每個回應都包含參考原始來源內容的引文。
- 您可以選取引文以直接導覽至相關的課程、模組或檔案。
- 引文可協助您驗證資訊，並在需要時探索其他內容。

### 串流回應

AI Assistant會在產生答案時逐步提供答案，因此您可以立即開始閱讀，而不需要等候完整回應載入。

### 引文和來源透明度

每個AI Assistant回應都包含直接連結至原始課程、模組或學習物件的引用。 引文可讓您：

- 選取內嵌引文號碼以跳至完全參照的區段。
- 選取回應底部的&#x200B;**顯示來源**，開啟完整的來源清單。
- 驗證資訊，並從權威來源探索其他內容。

> **重要**
> AI助理會根據管理員啟用的內容提供答案。 如果您沒有參考專案的存取權，當您嘗試開啟時，將會看到「不支援」訊息。


## 內建提示

AI助理包含內建提示，可幫助您快速開始處理常見問題和情境。 這些提示會引導您如何與助理互動，並示範您可以詢問的問題型別。

![學習者助理提供的內建提示](assets/built-in-prompt-new.png)

組織可以自訂內建提示，以反映其學習目標、角色、術語或特定使用案例。 管理員可與他們的客戶成功經理合作，以設定或更新內建提示。 在目前版本中，您無法直接在Adobe Learning Manager介面中自訂提示。

## 設定AI助理（管理員）

![啟用AI的學習者助理](assets/learner-ai-assistant-new.png)

管理員選取哪些&#x200B;**內部**&#x200B;目錄可以存取AI助理功能。 請確定您指派的目錄僅包含適用於AI回應和引用的學習內容，且這些目錄為&#x200B;**內部** （非&#x200B;**已共用**、**已取得**&#x200B;或&#x200B;**外部**）。

在設定「AI小幫手」之前，請確認您擁有管理員認證，並已識別目錄應該具有存取權。

### 設定AI助理存取權

若要啟用學習者AI小幫手：

1.以管理員身分登入Adobe Learning Manager。

2.從首頁中選取&#x200B;**設定**。
![在左窗格](assets/settings-menu.png)具有[設定]選項的管理員主控台

3.從&#x200B;**設定**&#x200B;功能表選取&#x200B;**學習者AI助理(Beta)**。
![管理員主控台在左窗格](assets/learner-assistant-ai-beta.png)顯示學習者AI助理選項

4.選取切換開關以啟用&#x200B;**學習者AI助理(Beta)**。
<!--![Administrators console displays the toggle enabled for Learner AI Assistant](assets/learner-assistant-toggle.png)-->
<!--5. Select one or more user groups from the **Eligible user groups** option.-->
<!--5. Select **Save** to apply the user group settings.-->

5.從&#x200B;**合格目錄**&#x200B;選項中選取一或多個目錄。

6.選取&#x200B;**儲存**&#x200B;以套用目錄設定。

>[!IMPORTANT]
>
>僅支援&#x200B;**內部**&#x200B;目錄。 如果選取了&#x200B;**共用**、**已取得**、**外部**&#x200B;或其他非內部目錄，即使其出現在&#x200B;**合格目錄**&#x200B;清單中，AI助理也不會顯示其內容。

## 啟動AI助理（學習者）

若要啟動AI助理：

1. 以學習者身分登入Adobe Learning Manager。

2. 在首頁上選取&#x200B;**詢問AI助理**。
   ![學習者首頁顯示「詢問AI小幫手」以選取並開啟「學習者AI小幫手」面板](assets/ask-ai-assistant.png)

3. 當&#x200B;**學習者AI助理**&#x200B;畫面出現時，請選取&#x200B;**開始使用**。
   ![選取[開始使用]以啟動學習者助理](assets/get-started-learner-assistant.png)

>[!NOTE]
>
>第一次啟動AI Assistant時，您必須先提供同意才能使用。 同意對話方塊只會在初次啟動時顯示。 對於所有後續啟動，您將直接帶到AI助理輸入提示。

4.在文字欄位中輸入提示。
<!-- ![Type prompt in the Learner Assistant](assets/type-prompt-new.png) -->

5.按&#x200B;**Enter**&#x200B;以接收回應。 檢閱您的答案、來源和建議。

您可以：

- 選取內嵌的引文編號以跳至確切的參考區段
- 選取回應底部的&#x200B;**顯示來源**，開啟完整的來源清單

![在回應中顯示來源](assets/show-sources-latest.png)

AI Assistant包含每個回應的引文，以顯示資訊的來源。 每個引文都直接連結至用來產生答案的原始課程、模組或學習物件。

您可以選取任何引文，以在Adobe Learning Manager中開啟課程頁面，並在內容中檢閱完整內容。 引用可協助您驗證資訊、探索其他詳細資訊，並繼續從權威來源學習。

## 透過搜尋存取AI助理

您也可以直接從搜尋列啟動AI助理。 在搜尋欄位中輸入您的問題，然後從顯示的選項中選取&#x200B;**詢問AI小幫手**。

![從搜尋列存取學習者助理](assets/learner-assistant-search-new.png)

## 提供對AI助理回應的意見回饋

您對AI助理(Beta)所產生回應的意見反應有助於改善其正確性、相關性和整體效能。

### 喜歡或不喜歡回應

- 選取&#x200B;**向上縮圖**，選擇您認為有幫助的回應，並選擇加入註解，然後選取&#x200B;**提交**。
- 選取&#x200B;**向下縮圖**，選擇回應沒有幫助的原因，新增任何註解，然後選取&#x200B;**提交**。

## 開始新的聊天

開始新的聊天可讓您開始新的交談，清除先前的上下文，以便助理可以聚焦於新主題，而不參考先前的互動。

若要清除目前的交談內容並重新開始，請在AI助理畫面中選取&#x200B;**新增交談**，然後選取&#x200B;**是**。

![在學習者助理中開始新的聊天](assets/start-new-chat.png)

AI Assistant為學習者提供快速的情境式答案，支援多種內容型別，並提供內嵌的透明度引文。 管理員可以控制存取權，確保AI Assistant根據組織需求量身打造，並增強學習體驗。


## 疑難排解AI助理問題

> **附註**
> 設定新目錄後，請等待4-5小時讓內容索引化，並可用於AI助理回應。

### 無權存取內容

**問題：**&#x200B;學習者可以存取AI小幫手，但會收到「我沒有此問題的答案」回應。

**可能的原因：**

- 學習者的目錄未包含在AI助理設定中。
- 與問題相關的內容不在所選目錄中，或目錄是空的。
- 學習者無法檢視相關內容。

**解決方案：**

- 驗證學習者的目錄存取權。
- 檢查在AI助理設定中啟用的目錄。
- 請確定這些目錄中存在相關內容。
- 新增內容後，請等候數小時，以便索引該內容。

### 無關緊要或低品質的答案

**問題：** AI小幫手提供的答案不符合問題或品質不佳。

**可能的原因：**

- 這個問題太寬泛或模稜兩可。
- 相關內容的中繼資料（說明、標籤）不佳。
- 內容結構使得擷取資訊變得困難。

**解決方案：**

- 鼓勵學習者提出更具體的問題。
- 檢閱並改善課程說明和中繼資料。
- 確保內容有清楚的標題和結構。
- 檢閱詳細的使用情況報告以識別模式。
- 考慮為常見問題集建立工作輔助。

### 超出範圍的問題

**問題：**&#x200B;學習者提出與訓練內容無關的問題。

**範例：**

- 一般知識問題（「誰是總裁？」）
- 個人意見（「您對X有何看法？」）
- 不適當的內容

AI Assistant僅根據指派的學習內容回答問題，不會回應超出範圍的查詢。
