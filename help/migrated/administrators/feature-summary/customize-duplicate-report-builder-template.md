# 自訂重複的報表建工具範本

## 概觀

複製範本後，你可以新增或移除欄位、更新篩選條件、重新命名欄位，並調整排序，然後再儲存和下載。 本文將涵蓋每個客製化步驟。

開始前，從範本&#x200B;**標籤複製一個範本**。

## 新增與移除欄位

1. 在編輯檢視中，找到包含你想新增欄位的資料集。
2. 在欄位名稱旁選擇 **+** 鍵。 該專欄會出現在報告畫布中。
3. 例如，要重複加同一欄，計算兩個不同的狀態值。 再次選擇 **該欄位的 +** 鍵。
4. 要移除欄位，請在畫布中選取該欄位，然後選擇移除圖示。

## **欄位重新排序與重命名**

1. 將一欄拖曳到畫布中想要的位置。 畫布中的順序與下載檔案中的欄位順序相符。
2. 要重新命名欄位，選擇其別名欄位，並在報告中輸入你希望出現的欄位標題名稱。

## 更新篩選器

1. 要更改現有的濾波器值，請選擇篩選列，編輯該值，然後確認。
2. 要新增篩選器，請選擇 **新增篩選器**。
3. 選擇你想篩選的欄位。
4. 選擇一位操作員。 可用的運算子依欄位的資料型態而異。
5. 輸入或選擇一個數值：

   * 對於字串欄位，請輸入搜尋並從建議中選擇。

   * 對於清單欄位，請從選取器中選擇一個或多個值。

   * 對於日期欄位，請輸入日期或選擇相對範圍，例如過去90天。

6. 要移除篩選器，請選擇該篩選器列上的刪除圖示。

## 更新排序

1. 選擇你想排序的欄位上的排序圖示。
2. 選擇 **上升** 或 **下降**。
3. 若要在其他欄位排序，請對每一欄重複此步驟。 第一個排序欄位為主要欄位;在平手欄位中會進行額外的排序。

>[!TIP]
>
>務必至少塗抹一種。 若不排序，不同報告下載時列序可能會有所不同。

## 儲存與下載

1. 選擇 **儲存** 以將變更儲存到 **報告** 標籤。
2. 選擇 **下載** 以產生報告。 當檔案準備好時，你會收到應用程式內的通知。
