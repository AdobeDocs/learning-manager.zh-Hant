# 設定

## 概觀

Adobe Learning Manager 提供詳細的管理員設定，讓您能設定並自訂學習管理系統，以符合組織的特定需求。 這些場景被組織成幾個主要類別，每個類別負責不同的行政功能。

### 基本設定

* 此區塊允許管理員設定必要的組織細節，包括國家/地區、時區或地區。
* 為學習物件配置回饋機制與評分系統。
* 影響整體平台行為與使用情況的各種一般設定。

更多資訊請參閱 [Adobe Learning Manager 的設定](/help/migrated/administrators/feature-summary/settings/basic-settings.md) 。

### 整合設定

* 設定包括 Adobe ID 與單一登入（SSO）在內的認證選項。
* 管理與外部資料系統的整合，以協助使用者配置、課程目錄及報告資料同步。
* 建立並管理與其他 Learning Manager 帳號的連結，以促進內容分享與協作學習計畫。

更多資訊請參閱 Adobe Learning Manager 的設定。

### 進階設定

* 建立並管理自訂標籤，用於整理與分類學習內容，讓學習者更容易找到相關教材。
* 設置並管理由講師主導的訓練教室地點，包括容量、設備及排程細節。
* 配置合規與群組成功儀表板。

更多資訊請參閱 Adobe Learning Manager 的設定。
