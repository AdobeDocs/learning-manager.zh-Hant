# 在 Adobe Learning Manager 中設定一鍵註冊

## 一鍵註冊的運作方式

一鍵註冊依賴兩個 Adobe Learning Manager （ALM） 功能協同運作：深層連結提供ALM中特定課程或學習物件的直接網址。 你可以將這些連結（直接指向模組的網址）嵌入電子郵件、內聯網入口網站或第三方應用程式中。 如果學習者點擊連結時尚未登入，ALM 會提示他們驗證，然後直接帶他們進入課程。

當學習者在工作中使用 Slack 或 Teams，或需要在出差、參加會議或銷售前快速進行兩分鐘複習時，這通常很有價值。他們可以立即存取內容並接受訓練。
與註冊相關的服務會自動將學習者註冊到課程，然後再透過深度連結啟動課程播放器。這消除了手動註冊步驟，避免了會中斷學習者的學習體驗。

當學習者選擇一鍵註冊連結時，ALM 會透過背景的 API 將他們註冊，然後透過深層連結將他們導向課程。 球場玩家會立刻打開。

>[!NOTE]
>
>註冊僅在課程層級進行，而非高階學習對象（學習路徑或認證）。

## 為模組產生深度連結

1. 以管理員身份登入 Adobe Learning Manager。
2. 在左側導覽欄選出 **課程** 。   ![](assets/one-click-enroll1.png)
3. 選擇課程
4. 選擇 **實**&#x200B;例。   ![](assets/one-click-enroll2.png)
5. 選擇&#x200B;**&#x200B;**&#x200B;你想複製的模組深層連結的模組區塊。模組細節會顯示在實例底部的展開區塊中。   ![](assets/one-click-enroll3.png)
6. 請前往你想複製連結的模組。   ![](assets/one-click-enroll4.png)
7. 選擇 **複製連結**。 深層連結現已複製。 這個深度連結是開啟課程中特定模組的連結。

你現在可以用這個深層連結把它寄給學習者。
