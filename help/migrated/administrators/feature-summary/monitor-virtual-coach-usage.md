# 監控虛擬教練的使用情況

查看虛擬教練使用數據、追蹤每月活躍用戶（MAU）信用消費，並可在Adobe Learning Manager中存取學習者表現報告。

## 為您的帳號啟用虛擬教練

虛擬教練可作為 Adobe Learning Manager 的附加元件使用。 購買後，設定會產生一組啟用金鑰，並透過電子郵件發送給帳戶管理員。

1. 以管理員身份登入 Adobe Learning Manager。
2. 從 **左側導覽窗格進入帳單** 頁面。
3. 在 **虛擬教練** 區，輸入你收到的啟動金鑰。
4. 選擇 **「申請」**。虛擬教練已啟用於你的帳號。   ![](assets/virtual-coach-037.png)

啟用後，你會收到應用程式內通知，確認該功能已上線。 四個範例角色扮演情境會自動加入內容庫，讓作者能立即開始。

>[!NOTE]
>
>啟用金鑰會在配置過程中自動產生，並透過電子郵件分享。 如果您沒有啟用金鑰，請聯絡您的 Adobe Learning 經理客戶成功經理（CSM）。

## 查看您的MAU信用餘額

每月活躍用戶（MAU）積分計算每月使用虛擬教練的獨立學習者數量。

1. 請前往 **帳單** 頁面。
2. 在 **虛擬教練** 區塊中，選擇 **查看使用詳情**。
3. 使用 **「選擇期間** 」下拉選單，選擇你想檢視的日期範圍。   ![](assets/virtual-coach-038.png)

   **整體使用**&#x200B;量表顯示：

   a. **可用：** 購買的總MAU點數\
   b. **使用：** 迄今為止消耗的積分\
   c. **剩餘** ：合約剩餘期間內可用的信用額度

   **每月使用**&#x200B;量表顯示各曆月獨立活躍學習者的數量。

4. 選擇 **下載詳細報告** 以匯出完整使用數據。

## MAU積分的消費方式

當學習者在曆月內完成虛擬教練課程時，會消耗一個MAU學分。 同一學習者在同一月份額外修課不會消耗額外學分。

| 劇本 | 消費的MAU數 |
|----------|--------------|
| 一名學員在一月完成五堂課 | 1 |
| 同一位學習者在一月和二月都使用 Virtual Coach | 2 次（每月 1 次） |
| 每年一月各有100名學員完成一堂課 | 100 |

_MAU學分會按每個月的獨立學習者計算，無論每位學習者完成多少次課程。_

合約期滿未使用的信用額度將失效且不會結轉。

### 範例一：單一學習者，多場課程

情境：Sarah 在一月完成五場虛擬教練課程。

* 一月的會議：5
* 消費的MAU數量：1
* 為什麼：Sarah 在一月內都被視為單一獨立用戶，無論她練習多少次。

### 範例二：同一位學習者，持續了好幾個月

情境：Sarah 在一月和二月都使用 Virtual Coach。

* 一月的會議：3
* 二月的場次：2
* 消費MAU：2（1月1 + 2月1）
* 為什麼：每個日曆月份都算得上獨立計算。

### 範例三：同一月份有多位學習者

情境：100 位銷售代表各完成一場 Virtual Coach 課程，時間為一月。

* 總計：100場
* MAU消費量：100
* 原因：每位獨特學習者在該月計入一個 MAU。

### 範例四：團隊練習

情境：你們50人的團隊全年使用虛擬教練。

| 月份 | 主動學習者 | 本月消費的MAU數 | 累積MAU數量 |
|------|----------------|--------------------------|-----------------|
| 年月 | 0 | 0 | 0 |
| 埃布魯阿里 | 5 人（5 人沒有練習） | 5 | 5 |
| 三月 | 0（全部練習增益50） | 0 | 45 |
| 四月 | 0 | 0 | 75 |

## 查看虛擬教練報告

AI 報告&#x200B;**頁面>>**&#x200B;**「管理**」報告&#x200B;**區塊提供組織內所有虛擬教練活動的使用與績效**&#x200B;資料。兩份報告可在虛擬教練&#x200B;**標題下**&#x200B;取得。

所有報告皆以 CSV 格式匯出。 報告產生可能需要數分鐘，視資料大小而定。

### 學習者使用摘要

包含所有學習者的每月使用數據。 利用此報告追蹤每月有多少學習者使用虛擬教練，監控 MAU 學分消費，並找出長期的參與趨勢。

### 會議詳情

包含過去90天所有學習者的課程層級資料。 利用本報告檢視個別課程分數、主題覆蓋率及學習者風格指標，並識別可能需要額外訓練或內容的技能缺口。

## 存取並下載報告

1. 以管理員身份登入 Adobe Learning Manager。
2. 在左側導覽欄選出 **「報告** 」。
3. 選擇 **AI 報告**。
4. 在 **虛擬教練** 區塊中，選擇你想下載的報告—— **學習者使用摘要** 或 **課程詳情**。   ![](assets/virtual-coach-039.png)
5. 當提示時選擇日期範圍，然後選擇 **繼續**。
6. 報告會自動下載為 CSV 檔案。
