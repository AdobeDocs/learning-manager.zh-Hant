# 在報表建工具中複製並重複使用報表

複製報告會複製其所有欄位、別名、依設定分組、聚合、篩選，並排序成新的可編輯報告。 當你想為不同的目錄、使用者群組或時期建立報告版本時，可以使用這個功能。

## 複製報告

1. 打開你想複製的報告。
2. 選擇&#x200B;**&#x200B;**>**重複動作**。一份報告副本會以編輯模式開啟，名稱為「[原始報告名稱]-副本」。 例如，跨經理的結構化績效檢視 - 文案
3. 請輸入報告的新名稱。
4. 做任何修改，例如更新篩選器、調整欄位或更改排序方式。
5. 選擇 **「儲存報告**」。

重複的報告會儲存在你的 **報告** 分頁，且與原始報告無關。

## 複製範本

你也可以從 **範本** 標籤複製範本。 複製範本會在報告&#x200B;**標籤中**&#x200B;建立一個新的可編輯報告。原始模板保持不變。

1. 選擇範本&#x200B;**&#x200B;**&#x200B;標籤。
2. 找到你想複製的範本。
3. 選擇 **重複**。   ![](assets/image049.png)
4. 輸入名稱、進行調整，然後選擇 **儲存報告**。

## 最佳實務

* 當你需要針對不同團隊、目錄或時期的變化時，可以複製一份經過充分測試的報告作為基礎。 這比從零開始建造更快，並確保結構一致。
* 立即將該副本重新命名，以反映其與原始內容的差異，例如「副總裁合規 – 亞太區」而非「合規報告副本」。
* 複製範本後，先更新名稱，再做其他更改，讓報告在報告標籤中能&#x200B;**&#x200B;**&#x200B;辨識。
