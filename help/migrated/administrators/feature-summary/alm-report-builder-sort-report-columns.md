# 在報表建工具中排序報表欄位

排序決定下載報告檔案中列的順序。 只要輸出一致重要，就套用排序。

## 新增排序

在這個例子中，你會知道完成率最高的課程。

1. 啟動報表建器並選擇 **建立報表**。
2. 請輸入報告的名稱與描述。
3. 請選擇以下欄位： `<dataset>:<column name>`a. 學習物件 - 學習物件名稱b. 學習物件 - 學習物件狀態c. 學習物件 - 完成次數
4. 在排序區塊中，選擇 **新增排序**。
5. 選擇 **學習物件 - 完成次數**。
6. 選擇 **排序順序：升序** 或 **降序**。   ![](assets/image032.jpg)
7. 選擇 **新增**。
8. 選擇 **儲存報告** 並選擇 **動作** > **下載** 以下載報告。

下載後的報告列出所有紀錄，依完成課程次數排序。

## 新增多欄排序

在這個範例中，你會產生一份報告，以衡量各經理的績效。

要依多欄排序：

1. 啟動 **報表建器** 並選擇 **建立報表**。
2. 請輸入報告的名稱與描述。
3. 請選擇以下欄位： `<dataset>:<column name>`a. 使用者 - 名稱b. 使用者-經理名稱c. 模組逐字稿 - 狀態d. 模組成績單 - 進度百分比
4. 加上以下總量：
a.依使用者群組 - 管理員名稱
b. 計數不同使用者 - 姓名
c. 計數 If=完成模組成績單 - 狀態
d. 平均模組成績單 - 進度百分比   ![](assets/image033.png)
5. 在 **排序** 區塊中，新增以下多欄排序：
a. 模組文字稿 - 狀態：下降中
b. 使用者 - 管理員名稱：升遷   ![](assets/image034.jpg)
6. 選擇儲存報告並選擇>下載行動以下載報告。

下載的報告提供依管理者分類的績效摘要，顯示不同的學習人數、基於狀態的註冊人數及平均進步百分比。 它強調不同管理團隊的參與程度與培訓進展。
